### Abstract
This project presents an automated attendance system using face recognition technology. The system utilizes OpenCV and LBPH (Local Binary Pattern Histogram) Face Recognizer to detect and identify faces in real-time. The implementation includes a user-friendly GUI interface built with Tkinter, secure password protection, and automated attendance tracking with CSV-based data storage. The system aims to eliminate manual attendance processes while ensuring accuracy and efficiency in attendance management.
